# 🚀 SkyForce - Arcade Shooter (Projeto Acadêmico)

Este é um projeto desenvolvido como parte do curso de **Sistemas de Informação** na faculdade **[Insira o Nome da Sua Faculdade Aqui]**. O objetivo é aplicar conceitos de Programação Orientada a Objetos (POO), gerenciamento de estados de jogo e interface gráfica multi-plataforma.

SkyForce é um shooter espacial (Shoot 'em up) feito em **C#** com **Avalonia UI**, focado em jogabilidade clássica, precisão técnica e combate estratégico contra chefes.

## 🎮 Guia de Comandos

Para jogar, utilize os seguintes comandos no seu teclado:

| Tecla | Ação |
| :--- | :--- |
| **Setas Direcionais** | Movimentam a nave pela tela |
| **Tecla L** | Dispara o tiro padrão (Laser) |
| **Tecla K (Segurar)** | **Movimento de Precisão:** Reduz a velocidade da nave para desvios delicados |
| **Tecla J** | **Ataque Especial:** Exclusivo do piloto **RoCoCop** (Consome 700 pontos) |
| **Enter** | Inicia o jogo, confirma seleção de piloto ou reinicia |
| **Esc** | Volta para o Menu Principal ou sai da tela atual |

## 🛠️ Detalhes de Balanceamento dos Pilotos

O jogo foi projetado com uma mecânica de risco e recompensa entre os dois pilotos disponíveis, exigindo estratégias diferentes:

1. **RoCoCop:** Possui o **Ataque Especial** (tecla J) que limpa a tela, mas seu dano base é menor (10). Ideal para controle de área contra hordas de inimigos.
2. **Sudo&Mudo:** Não possui ataque especial, porém compensa com um **dano massivo contra o Boss** (Causa 50 de dano por tiro, o dobro do seu dano base, contra apenas 10 do RoCoCop). É o especialista em eliminar alvos pesados.
