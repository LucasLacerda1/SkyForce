# 🚀 SkyForce - Arcade Shooter

SkyForce é um jogo de tiro espacial (Shoot 'em up) desenvolvido em **C#** utilizando o framework **Avalonia UI**. O projeto foca em jogabilidade fluida, combate contra chefes e uma experiência clássica de arcade com gráficos 2D.

## 🎮 Funcionalidades

- **Seleção de Pilotos:** Escolha entre dois personagens com atributos diferentes:
    - **RoCoCop:** Tiro rápido e cadenciado.
    - **Sudo&Mudo:** Dano massivo (2x dano contra Chefes), ideal para jogadores estratégicos.
- **Sistema de Pontuação:** Score dinâmico com animações de escala.
- **Ataque Especial:** Gaste 700 pontos para disparar um projétil tático que limpa a tela.
- **Batalha de Chefe:** Enfrente um Boss desafiador com 800 HP e padrões de tiro únicos.
- **Áudio Imersivo:** Mixagem de som balanceada entre trilha sonora e efeitos de explosão/tiros.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C# (.NET 8.0+)
- **Framework de UI:** [Avalonia UI](https://avaloniaui.net/) (Multiplataforma)
- **Áudio:** [NetCoreAudio](https://github.com/pdelvo/NetCoreAudio)
- **IDE Recomendada:** JetBrains Rider ou Visual Studio
